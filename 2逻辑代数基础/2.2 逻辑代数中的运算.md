# 一、三种基本逻辑
## 1.与 运算
### (1)算符：
“ · ”（或者X、∧、∩、AND)
### (2)运算规则：
==必须都是1，结果才为1==
0·0=0  1·0=0  0·1=0  1·1=1
### (3)逻辑表达式：
F = A · B
### (4)逻辑符号
![[与运算逻辑符号.png]]

## 2.或 运算
### (1)算符
“ + ”（或者∨、∪、OR）
### (2)运算规则
==有 1 ，结果就为 1==
0 + 0 = 0     1 + 0 = 1      0 + 1 = 1      1 + 1 = 1 
### (3)逻辑表达式：
F = A + B
### (4)逻辑符号
![[或运算逻辑符号.png]]

## 3.非 运算
### (1)算符
“ — ”
### (2)运算规则
$\overline{0}=1$   $\overline{1}=0$ 
### (3)逻辑表达式：
$F=\overline{A}$
### (4)逻辑符号
![[非运算逻辑符号.png]]
